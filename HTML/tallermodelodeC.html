Taller práctico HTML y CSS

1. Uso de la etiqueta mark

La etiqueta mark sirve para resaltar la información mas importante ya que se debe reconocer que en algunos casos solo se usan para resaltar una palabra y ya.

Normalmente el navegador muestra el texto resaltado con un fondo negro para llamar la atención.

Podemos usar mark para destacar conceptos, fechas o palabras importantes.

Funciona envolviendo el texto que queremos remarcar dentro de la etiqueta <mark>.

Explicación: La etiqueta mark es útil cuando queremos que un fragmento de texto se destaque visualmente. Esto no solo ayuda al lector a identificar información relevante rápidamente, sino que también es semánticamente correcto, ya que indica la importancia del contenido resaltado. Además, se puede personalizar su apariencia con CSS cambiando color de fondo, color de texto o estilo de borde, manteniendo así coherencia con el diseño de la página.

2. Diferentes displays

block
inline
inline-block
none

block: ocupa toda la línea.
inline: no respeta width ni height y se coloca en línea.
inline-block: permite tamaño y se coloca en línea.
none: oculta completamente el elemento.
